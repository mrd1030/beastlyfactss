# Gear Gaps: Species-Aware Breakdown (Full Pass)

Every remaining unlinked cost item used across 2+ guides (55 total), individually assessed:
is it a real Amazon-linkable product at all, and if so, does one product genuinely cover
every guide that lists it, or do different guides actually need physically different
products? Superseeds the earlier hand-picked 6-item version of this file - those 6 are
folded back in below with a more rigorous pass (a few came out split more finely once
actually reasoned through, e.g. chew toys).

**How to use this:** each `## ` item below is one former "used in N guides" checklist
entry. Check off each bullet once you've sourced and linked that specific variant.

**Bottom line:** of the 55 items, **30 are correctly non-linkable** (vet visits,
electricity, perishable food, or a bundle phrase combining products already linked
separately), 16 genuinely need only **one** product each, and 9 need to be **split** into
2-4 distinct products apiece. Net: **25 real product categories, ~41 individual products**
to source across those 25.
